# PSI Builder 📈

A Python/Flask-based paper trading application designed to simulate live-market trading strategies without financial risk. Built as an MSc final-year project.

## Features
* **Live Market Data:** Integrates with live broker APIs (Dhan, Angel One) for real-time tick data.
* **Virtual Trading:** Accurate virtual order execution and margin calculations.
* **Predictive Analytics:** Built-in linear regression models to forecast pricing trends.
* **Algorithmic Support:** Webhook integrations for automated trading strategies.
* **Interactive Dashboard:** Real-time portfolio tracking and historical trade logging.

## Tech Stack
* **Backend:** Python, Flask
* **Database:** MySQL
* **Machine Learning:** Scikit-learn, Pandas, NumPy
* **Frontend:** HTML, CSS, JavaScript (Jinja2)
